# Git命令

> #### 把文件放进缓存区
>
> ``git add .``
>
> #### 查看状态  
>
> ``git status``
>
> #### 提交
>
> ``git commit -m""``
>
> #### 推送进仓库
>
> ``git push -u origin(别名) maste(分支)``
>
> ``强推：git push -f -u origin(别名) maste(分支)``
>
> #### 下拉
>
> ``git pull origin(别名) master:master``
>
> #### 查看别名与远程仓库
>
> ``git remote -v``
>
> #### 获取远程仓库
>
> ``git remote add origin(别名) URL(地址)``
>
> #### 初始化git
>
> ``git init``
>
> #### 切换分支也可操作文件
>
> ``git checkout file(文件名或分支名)``
